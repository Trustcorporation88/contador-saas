# 📚 Documentação de Serviços

Bem-vindo à documentação dos serviços oferecidos pelo escritório. Esta pasta reúne guias práticos sobre os principais processos **fiscais**, **financeiros** e **contábeis** que utilizamos no dia a dia.

## 🧭 Visão Geral

Cada documento descreve um serviço de forma objetiva, respondendo a quatro perguntas centrais:

1. **O que é** o serviço?
2. **Quando** devo utilizá-lo?
3. **Quais informações** preciso reunir antes de começar?
4. **Quais erros** evitar e **quanto tempo** leva?

O objetivo é servir tanto como onboarding para novos colaboradores quanto como referência rápida para clientes (contadores parceiros e pequenas empresas).

## 📖 Como usar esta documentação

- Procure pelo serviço desejado no índice abaixo.
- Leia a seção **"O que é?"** e **"Quando usar?"** para confirmar que é o documento certo.
- Antes de iniciar a execução, revise a lista de **Variáveis Necessárias** — ela funciona como checklist.
- Consulte **Erros Comuns** e **FAQ** quando algo não sair como esperado.
- Para uma visão consolidada de todos os serviços, veja o [GUIA-COMPLETO-SERVICOS.md](./GUIA-COMPLETO-SERVICOS.md).
- Para criar um novo documento de serviço, copie o [_TEMPLATE.md](./_TEMPLATE.md).
